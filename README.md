# ChondroGenis 🧬

ChondroGenis is an open-source computational biology and molecular modeling platform 
dedicated to simulating the reactivation of epiphyseal (growth) plates in adults. 

The project focuses on accelerating vertical progress in tissue engineering and 
regenerative medicine by in-silico modeling of chondrogenesis, extracellular matrix 
remodeling, and target peptide-receptor interactions (such as CNP, PTHrP, and FGFR3 pathways).
